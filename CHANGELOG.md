# Changelog

All notable changes to Fedora DNF GUI Tool are documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-03-12

### Added

- **DNF Package Management** — Install, remove, update packages; browse installed with search/filter
- **Flatpak Manager** — Browse installed Flatpaks, search Flathub, install/remove/update apps
- **System Info** — Live dashboard with Fedora version, kernel, KDE Plasma, CPU, GPU, RAM, disk usage
- **Quick Tools** — 20+ one-click actions: RPM Fusion, codecs, dev tools, VS Code, fwupd, popular apps
- **Repository Manager** — View, enable/disable repos; add COPR repositories
- **Transaction History** — Browse DNF history with undo capability
- **Auto-update** — Check for app updates from GitHub releases
- **Live Terminal** — Real-time output from all package operations
- **Dark theme** — Polkit integration for privileged operations
